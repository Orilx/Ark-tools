# Ark-tools

一些方便提取和处理《明日方舟》游戏资源的小工具

## 文件说明

- `download_res.py` 检查并自动下载更新的游戏资源
- `gen_avg.py` 解包剧情人物立绘差分资源
- `gen_face.py` 把解包好的立绘差分拼在一起 
- `equip_data.py` 解析模组数据
